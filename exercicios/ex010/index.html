Joga a âncora, marujo

toda vez que voçê clica em um link, para ir para outra pagina ou até fazer um download, voçê está interegindo com um hyperlink

para criar um hyperlink, devemos criar âncoras com a tag <a> e seu principal atributo é o href, que cria uma referencia hipertexto

acesse o site da amazon

outra tag bem útil da tag âncora é o hreflang, que permite indicar qual é o idioma principal do site onde o link esta desviando o fluxo de navegação.

acesse o site da amazon
